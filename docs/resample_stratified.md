# Stratified resampling
## Usage
* `alpha = resample_stratified(w)`
 
## Description
Stratified resampling, returns randomized indices `alpha` such that
Pr(alpha) = w(alpha).
 
## Input
* `w`: Probabilities.
 
## Output
* `alpha`: The resampled indices.
